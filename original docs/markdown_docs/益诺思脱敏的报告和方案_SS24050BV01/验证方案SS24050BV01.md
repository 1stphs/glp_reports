材料和方法

方法SS24050-BA-001描述了本方法使用的仪器、软件、溶剂和试剂、基质以及配制和处理步骤。概述如下：体积为50 µL的猪EDTA-K2血浆样品采用蛋白沉淀法进行样品提取。810Z00-T将以Dexamethasone为内标用LC-MS/MS进行检测和定量。

方法学验证内容和接受标准

系统适用性

系统适用性考察是在分析批开始前进行以确保系统正常运转。处理至少1份系统适用性样品并连续进样至少3次，以最后3次进样来评估系统适用性。

接受标准：在同一分析批中待测物及内标保留时间的变异系数（%CV）应不超出5.0%，待测物与内标色谱峰面积比值的变异系数（%CV）应不超出15.0%。

残留

残留通过在定量上限（ULOQ）样品后面进样双空白样品（不含待测物和内标）来进行评估。

接受标准：双空白样品中待测物保留时间处响应不超过同一分析批标曲符合接受标准的定量下限（LLOQ）待测物响应（均值）的20.0%，内标保留时间处响应不超过同一分析批符合接受标准的标准曲线定量下限内标响应（均值）的5.0%。

若残留无法满足接受标准，在方法学验证及样品分析过程中根据SOP-EXO-ABA-004对残留不满足接受标准批次的每一个样品受到上一个样品残留的影响进行评价，具体方法如下：

计算残留因子（若分析批中有多个，则选用最小的一个）：

残留因子 = 1 / （双空白样品中的峰面积/前一个ULOQ中的峰面积）

计算对下一个样品的影响率：

影响率 = 上一个样品的峰面积 / 残留因子 / 下一个样品的峰面积 * 100%

若影响率不高于5%，可认为实际发生的残留不会影响下一个样品的定量；若残留影响率大于5%，则说明下一个样品的定量结果受到了残留的影响，样品应进行相应处理：

质控样品：不论样品的准确度是否在85%~115%范围内，均按不符合接受标准计。

方法学验证考察项样品：该样品的结果不予采纳。若因残留评价失败导致考察项的统计数不够，则需重新做考察项。

若分析方法的残留不符合接受标准，可在分析方法中注明高浓度质控及ULOQ样品后需添加双空白样品。如有必要，可进一步优化分析方法解决残留问题。

选择性

选择性是区分目标分析物和内标与基质的内源性组份的能力。通过考察6个不同个体的正常空白基质、至少1个个体的溶血空白基质（加入溶血全血至未溶血的血浆中制备成2%的溶血血浆）进行不加内标的空白样品测定来评价选择性。

接受标准：空白基质中的干扰组份在待测物保留时间处的响应值不得超过符合接受标准的标准曲线定量下限峰面积（均值）的20.0%，内标保留时间处响应值不得超过符合接受标准的标准曲线定量下限内标峰面积（均值）的5.0%。若某个来源（批次）的空白基质样品不符合该接受标准，将采用同样的方法评估额外的3个不同来源（批次）空白基质对待测物和内标的干扰程度。若超过1个来源（批次）或者额外增加考察的空白基质样品不符合该接受标准，将进行调查或优化方法，项目负责人根据调查结果决定重复该实验、增加额外空白基质测试、修正该分析方法并重新进行方法学验证等。

特异性

特异性是生物分析方法检测和区分待测物与其他物质的能力。

待测物与内标间的干扰

 待测物对内标的干扰

分别平行制备、处理并分析三份仅含单个待测物且不加内标的定量上限浓度样品。

接受标准：仅含单个待测物的样品内标保留时间处的平均峰面积应不大于同一分析批中符合接受标准的标准曲线定量下限内标峰面积（均值）的5.0%。

 内标对待测物的干扰

分别平行制备、处理并分析三份仅含单个内标且不加待测物的样品，内标浓度与方法验证时所用内标浓度一致。

接受标准：待测物保留时间处的平均峰面积应不大于同一分析批中符合接受标准的标准曲线定量下限峰面积（均值）的20.0%。

基质效应

每个浓度基质效应系指生物基质中存在的组份对分析物的离子化所产生的抑制或增强作用。

分别取6个不同个体的空白基质，每个个体分别制备低浓度和高浓度质控样品进行分析，每个个体三个重复。

接受标准：每个个体的基质，每个浓度水平测得的浓度均值相对于标示浓度的偏差在±15.0%范围内，且变异系数（%CV）不大于15.0%。

标准曲线及范围

810Z00-T的标准曲线浓度范围为50.000 pg/mL到25000.000 pg/mL；标准曲线样品的浓度分别为：50.000、100.000、250.000、1000.000、2500.000、12500.000、20000.000、25000.000 pg/mL。以待测物浓度为横坐标，待测物与内标化合物色谱峰的峰面积比值为纵坐标，用加权最小二乘法进行回归运算（权重为1/X2），求得回归方程，即为标准曲线。在方法验证中评价不同天内的至少3个独立批的标准曲线。

接受标准：

标准曲线各浓度点的回算值与标示值之间的偏差须在±15.0%以内（定量下限处的偏差在±20.0%范围内）。

标准曲线的R2不应低于0.9800。

75%的非零浓度点且至少有六个浓度点（包括LLOQ和ULOQ）用于标准曲线的计算，且满足以上标准。

每个浓度点含有2个标准曲线样品的情况下，必须满足每个浓度点至少50%的样品满足接受标准。

批内/批间准确度及精密度

取五个不同浓度的质控样品（810Z00-T浓度分别为50.000 pg/mL（定量下限）、150.000 pg/mL （低）、1125.000 pg/mL（几何均值）、7500.000 pg/mL（中）和18750.000 pg/mL（高），考察批内/批间准确度和精密度。每个分析批每个浓度平行处理6份，需新鲜配制准确度精密度样品。 批内准确度和精密度通过每个分析批的质控样品进行评价；批间准确度和精密度通过三个独立分析批（至少2天进行）中的质控样品进行评价。

报告的方法验证数据以及准确度和精密度的测定应包括所有获得的结果，包括不符合接受标准的单个质控样品，已被记录的明显错误情况除外。同时还要提交每个分析批的批内准确度和精密度数据。

如果准确度和精密度考察项失败，确定失败的可能原因，并在下一个验证批次重新考察。如果下一个验证批次也失败，则应在重新启动验证之前修改该方法。如果超过一个分析批失败且原因不明确，分析步骤需要相应进行调整，方法学重新验证。

接受标准：

每一浓度水平下批内和批间准确度偏差应在±15.0%范围内（定量下限处的偏差在±20.0%范围内）。

每一浓度水平下批内和批间精密度应不超过15.0%（LLOQ样品不超过20.0%）。

质控样品

每个分析批至少包含低、几何均值、中、高4个浓度的质控样品（每个浓度水平至少2份）。含准确度精密度的分析批，低、几何均值、中和高浓度的准确度与精密度样品也作为质控样品。

接受标准：每个分析批至少2/3的质控样品且每个浓度水平至少50%的样品准确度偏差应在±15.0%范围内。

稀释可靠性

采用配制浓度高于ULOQ的DQC样品（含810Z00-T浓度为375000.000 pg/mL）进行稀释可靠性考察。用空白猪血浆20倍稀释该样品，平行稀释6份，稀释20倍后810Z00-T浓度为18750.000 pg/mL 。随行标准曲线样品对稀释后样品进行测定。

接受标准：稀释因子校正后的平均回算浓度的准确度须在85.0%~115.0%以内，变异系数（%CV）不大于15.0%。

冻融稳定性：分别将低、高浓度的质控样品经过5次（考察次数可根据实际情况调整）冻融（-70~-90ºC /室温白光）循环（质控样品须被冷冻至少12小时后再充分融化）。各浓度水平平行测定6份。

处理后样品稳定性：经处理后的样品（可采用某一个分析批已成功进样的准确度和精密度样品），放置在自动进样器中或与自动进样器相同温度的环境中一段时间后（24小时至144小时，考察时间可根据实际情况调整）分析。

长期稳定性：分别将低、高浓度的质控样品储存于超低温冰箱（-70~-90ºC ）中（1个月，考察时间可根据实际情况调整）。各浓度水平平行测定6份。样品稳定性至少应涵盖从第一个样品采集至样品分析结束的时间段。如有需要，可以评估更长的周期。

稳定性考察不符合接受标准时的处理办法

如果稳定性考察中其中一个或者两个浓度的测得浓度均值不符合接受标准，稳定性考察可重复测定或者直接缩短考察时间。如果重复的稳定性考察仍然不能满足接受标准，报告质控样品在该条件下不稳定。可以考察更短的储存时间。如果第二次稳定性考察满足接受标准，需要进行第三次稳定性考察。如果第三次稳定性考察确认了第二次稳定性考察的结果，第三次稳定性考察的结果作为最终的稳定性数据进行报告。如果第三次稳定性考察不满足接受标准，报告稳定性考察失败。考察更短的储存时间。

如果稳定性考察中其中一个或者两个浓度的变异系数（%CV）大于15.0%，稳定性考察需要重复测定一次。

溶血效应

使用模拟的溶血血浆样品（加入溶血全血至未溶血的血浆中制备成2%的溶血血浆）制备低浓度和高浓度质控样品进行评估。每个浓度平行测定3份样品。以正常血浆配制的标准曲线样品测定上述溶血质控样品，并随行正常血浆配制的质控样品。

接受标准：测得的浓度均值相对于标示浓度在85.0%~115.0%范围内，且变异系数（%CV）不大于15.0%。

如果溶血实验的质控样品不符合接受标准，应该调整分析方法。在分析方法调整和优化后，如溶血实验的质控样品仍然不符合常规接受标准，这个方法可以使用，但溶血效应的影响应该记录在项目的实验记录和验证报告中。

分析批最大进样数

为评估分析批最大进样数及方法的耐用性，应至少有一个验证分析批的样品数量接近于样品分析时预估的一个分析批中包括的样品数量。此分析批中可以包括更多的验证样品，重复进样质控样品或者添加内标的空白基质。

接受标准：该分析批标曲和质控样品满足分析批的接受标准。
