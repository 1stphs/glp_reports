## Summary
Final bioanalytical validation report for study SS24050BV01. Validates the LC-MS/MS method for determining 810Z00-T in Bama Minipig EDTA-K2 plasma. Confirms the method meets GLP requirements with a linear range of 50.000~25000.000 pg/mL. Covers all validation parameters including accuracy, precision, selectivity, recovery, and comprehensive stability.

批内/批间准确度及精密度

批内/批间准确度及精密度按验证方案考察，根据质控样品测定结果计算准确度及精密度，结果见表9。

除准确度及精密度分析批的质控样品按验证方案测定，除准确度及精密度分析批的质控样品结果见表10。结果表明，所有接受批的质控样品均符合分析批接受标准。

回收率

待测物的提取回收率

待测物的提取回收率按验证方案测定，结果见表11。结果表明，以峰面积比计算的本方法测定低、中、高浓度810Z00-T每一浓度的提取回收率平均值在91.3%~99.9%之间，变异系数（%CV）均不大于10.0%；总体变异系数（%CV）为4.8%。

内标的提取回收率

内标的提取回收率按验证方案测定，结果见表12。结果表明，以峰面积比计算的本方法测量的内标的提取回收率平均值为101.9%，变异系数（%CV）为5.3%。

 稀释可靠性

稀释可靠性按验证方案测定，结果见表13。结果表明巴马小型猪血浆样品经空白血浆稀释20倍后测定不影响结果的准确性。

 稳定性

标准品储备液/工作液稳定性

标准品储备液稳定性结果参考22037MV07。结果表明810Z00-T储备液可在室温白光条件下稳定至少16小时，在超低温冰箱（-70~-90ºC）稳定至少131天。

标准品工作液稳定性按验证方案测定，结果见表15和表17。结果表明810Z00-T工作液可在室温白光条件下稳定至少15小时，在超低温冰箱（-70~-90ºC）稳定至少69天。

内标储备液/工作液稳定性

内标储备液稳定性结果参考22037MV07。结果表明内标储备液可在室温白光条件下稳定至少16小时，在超低温冰箱（-70~-90ºC）稳定至少131天。

内标工作液稳定性按验证方案测定，结果见表15和表17。结果表明内标工作液可在室温白光条件下稳定至少15小时，在超低温冰箱（-70~-90ºC）稳定至少69天。

质控样品稳定性

短期稳定性

短期稳定性按验证方案测定，测定结果见表18。结果表明，巴马小型猪血浆样品中810Z00-T均可在室温白光条件下稳定至少17小时。

冻融稳定性

冻融稳定性按验证方案测定，结果见表19。结果表明，巴马小型猪血浆样品中810Z00-T可稳定至少4次（-70~-90 ºC/室温白光）冻融循环。

处理后样品的稳定性

处理后样品的稳定性按验证方案测定，结果见表20。结果表明，处理后的巴马小型猪血浆样品中810Z00-T在自动进样器温度（5ºC）下可稳定至少103小时。

长期稳定性

长期稳定性按验证方案测定，结果见表21。结果表明，巴马小型猪血浆样品中810Z00-T可在超低温冰箱（-70~-90 ºC）稳定至少82天。

全血稳定性

全血稳定性按验证方案测定，结果见表22。结果表明，巴马小型猪全血样品中810Z00-T在室温白光放置可稳定至少1小时。

 溶血效应

溶血效应按验证方案测定，结果见表23。结果表明，溶血对本方法定量测定巴马小型猪血浆中810Z00-T无影响。

 分析批最大进样数

分析批最大进样数按验证方案考察，本验证实验中最大进样针数为136针。

结论

本实验对分析方法“LC-MS/MS法测定猪EDTA-K2血浆中810Z00-T（RESS24050-B）浓度”进行了完全验证。本实验成功验证了分析方法的系统适用性，残留，选择性，特异性，基质效应，标准曲线及范围，批内/批间准确度及精密度，提取回收率，稀释可靠性，储备液/工作液稳定性，样品短期稳定性，样品冻融稳定性，处理后样品稳定性，样品长期稳定性，全血稳定性，溶血效应和分析批最大进样针数。

综上，本方法可以用于测定巴马小型猪EDTA-K2血浆中的810Z00-T的浓度。
