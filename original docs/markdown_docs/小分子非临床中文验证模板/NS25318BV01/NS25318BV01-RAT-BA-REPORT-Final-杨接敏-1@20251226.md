验证目的

本验证的目的是验证LC-MS/MS方法测定SD大鼠EDTA-K2血浆样品中待测物A的浓度测定方法（NS25318-BA-001）。并对该方法进行系统适用性、残留、选择性、特异性、基质效应、标准曲线及范围、批内/批间准确度及精密度、提取回收率、稀释可靠性、稳定性、溶血效应和分析批最大进样数等全面方法学验证，以支持非临床试验生物分析。

批内/批间准确度及精密度

批内/批间准确度及精密度按验证方案考察，根据质控样品测定结果计算准确度及精密度，结果见表9。

除LLOQ QC以外的所有浓度质控的批内准确度偏差范围为0.2%~11.2%，批内精密度最大值为8.4%；批间准确度偏差范围为2.8%~6.8%，批间精密度最大值为6.1%。

LLOQ QC的批内准确度偏差范围为-2.0%~15.0%，批内精密度最大值为8.7%；批间准确度偏差为4.0%，批间精密度为9.6%。

结果表明，质控样品的批内、批间精密度和准确度均符合接受标准。

质控样品

除准确度及精密度分析批的质控样品按验证方案测定，除准确度及精密度分析批的质控样品结果见表10。结果表明，所有接受批的质控样品均符合分析批接受标准。

回收率

待测物的提取回收率

待测物的提取回收率按验证方案测定，结果见表11。结果表明，以峰面积比计算的本方法测定低、中、高浓度待测物A每一浓度的提取回收率平均值在62.8%~65.5%之间，变异系数（%CV）均不大于6.8%；总体变异系数（%CV）为2.3%。

内标的提取回收率

由于使用稳定同位素标记的内标，内标的提取回收率参照待测物的提取回收率。

 稀释可靠性

稀释可靠性按验证方案测定，结果见表13。结果表明SD大鼠血浆样品经空白基质稀释20倍后测定不影响结果的准确性。

 稳定性

标准品储备液/工作液稳定性

标准品储备液稳定性按验证方案测定，结果见表14和表16。结果表明待测物A储备液可在室温白光条件下稳定至少22小时，在超低温冰箱（-70~-90 ºC）稳定至少88天。

标准品工作液稳定性按验证方案测定，结果见表15和表17。结果表明经待测物A工作液可在室温白光条件下稳定至少22小时，在超低温冰箱（-70~-90 ºC）稳定至少88天。

内标储备液/工作液稳定性

由于使用稳定同位素标记的内标，内标储备液和工作溶液的稳定性不做考察。防止同位素交换产生的影响，于分析批1（特异性）、分析批8和9评估待测物与内标之间的干扰，结果表明待测物与内标的干扰符合接受标准。

质控样品稳定性

短期稳定性

短期稳定性按验证方案测定，测定结果见表18。结果表明，SD大鼠血浆样品中待测物A可在冰水浴白光条件下稳定至少16小时。

冻融稳定性

冻融稳定性按验证方案测定，结果见表19。结果表明，SD大鼠血浆样品中待测物A可稳定至少4次（-70~-90 ºC/冰水浴白光）冻融循环。

处理后样品的稳定性

处理后样品的稳定性按验证方案测定，结果见表20。结果表明，处理后的SD大鼠血浆样品中待测物A在自动进样器温度（5ºC）下可稳定至少78小时。

处理后样品再进样重现性

处理后样品再进样重现性按验证方案测定，结果见表24。结果表明，处理后的SD大鼠血浆样品中待测物A在自动进样器温度（5ºC）下至少65小时后再进样结果可重现。

长期稳定性

长期稳定性按验证方案测定，结果见表21。结果表明，SD大鼠血浆样品中待测物A可在超低温冰箱（-70~-90 ºC）稳定至少95天。

全血稳定性

全血稳定性按验证方案测定，结果见表22。结果表明，SD大鼠全血样品中待测物A在冰水浴白光放置可稳定至少1.5小时。

溶血效应

溶血效应按验证方案测定，结果见表23。结果表明，2%溶血符合接受标准。

 分析批最大进样数

分析批最大进样数按验证方案考察，本验证实验中最大进样针数为118针。

结论

对分析方法“LC-MS/MS方法测定SD大鼠EDTA-K2血浆样品中待测物A(RENS25318)的浓度”进行了完全验证。本实验成功验证了分析方法的系统适用性，残留，选择性，特异性，基质效应，标准曲线及范围，批内/批间准确度及精密度，提取回收率，稀释可靠性，标准品储备液/工作液稳定性，样品短期稳定性，样品冻融稳定性，处理后样品稳定性，样品长期稳定性，全血稳定性，溶血效应和分析批最大进样针数。

综上，本方法可以用于测定SD大鼠EDTA-K2血浆中的待测物A的浓度。
