回收率

待测物的提取回收率

待测物的提取回收率按验证方案测定，结果见表11。结果表明，以峰面积比计算的本方法测定低、中、高浓度待测物A（游离型）反义链每一浓度的提取回收率平均值在72.8%~88.9%之间，变异系数（%CV）均不大于6.3%；总体变异系数（%CV）为11.1%。待测物A（游离型）正义链每一浓度的提取回收率均值在83.0%~90.3%之间，变异系数（%CV）均不大于6.5%；总体变异系数（%CV）为4.3%。

 稳定性

标准品储备液/工作液稳定性

标准品储备液稳定性按验证方案测定，结果见表14和表16。结果表明待测物A（游离型）反义链和正义链标准品储备液可在室温白光条件下稳定至少22小时，在超低温冰箱（-70~-90 ºC）稳定至少103天。

标准品工作液稳定性按验证方案测定，结果见表15和表17。结果表明待测物A（游离型）反义链和正义链工作液均可在室温白光条件下稳定至少22小时，在超低温冰箱（-70~-90 ºC）稳定至少103天。

内标储备液/工作液稳定性

内标储备液稳定性按验证方案测定，结果见表14和表16。结果表明内标储备液可在室温白光条件下稳定至少22小时，可在超低温冰箱（-70~-90 ºC）稳定至少103天。

内标工作液稳定性按验证方案测定，结果见表15和表17。结果表明内标工作液可在室温白光条件下稳定至少22小时，可在超低温冰箱（-70~-90 ºC）稳定至少103天。

质控样品稳定性

短期稳定性

短期稳定性按验证方案测定，测定结果见表18。结果表明，SD大鼠血浆样品中待测物A（游离型）反义链和正义链均可在室温白光条件下稳定至少17小时。

冻融稳定性

冻融稳定性按验证方案测定，结果见表19。结果表明，SD大鼠血浆样品待测物A（游离型）反义链和正义链均可稳定至少5次（-70~-90 ºC/室温白光）冻融循环。

处理后样品的稳定性

处理后样品的稳定性按验证方案测定，结果见表20。结果表明，处理后的SD大鼠血浆样品中待测物A（游离型）反义链和正义链在自动进样器温度（5ºC）下均可稳定至少151小时。

长期稳定性

长期稳定性按验证方案测定，结果见表21。结果表明，SD大鼠血浆样品中待测物A（游离型）反义链和正义链可在超低温冰箱（-70~-90 ºC）均可稳定至少106天。

全血稳定性

全血稳定性按验证方案测定，结果见表22。结果表明，SD大鼠全血样品中待测物A（游离型）反义链和正义链在室温白光放置均可稳定至少2小时。

 溶血效应

溶血效应按验证方案测定，结果见表23。结果表明，溶血对本方法定量测定SD大鼠血浆中待测物A（游离型）反义链和正义链无影响。

结论

本实验对分析方法“LC-MS/MS方法测定SD大鼠EDTA-K2血浆样品中待测物A（游离型）（RESS25071）反义链和正义链的浓度”进行了完全验证。本实验成功验证了分析方法的系统适用性，残留，选择性，特异性，基质效应，标准曲线及范围，批内/批间准确度及精密度，提取回收率，稀释可靠性，储备液/工作液稳定性，样品短期稳定性，样品冻融稳定性，处理后样品稳定性，样品长期稳定性，全血稳定性，溶血效应和分析批最大进样针数。

综上，本方法可以用于测定SD大鼠EDTA-K2血浆中的待测物A（游离型）反义链和正义链的浓度。
