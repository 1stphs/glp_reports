方法SS5255-BA-003描述了本方法使用的仪器、软件、溶剂和试剂、基质以及配制和处理步骤。概述如下：体积为10 µL的BALB/c裸鼠EDTA-K2血浆样品采用蛋白沉淀法进行样品提取。待测物A和待测物B（待测物B-1）将分别以内标A和内标B为内标用LC-MS/MS进行检测和定量。

分析系统

储备液的配制

精密称取适量Dxd标准品于合适容器中，加入适量稀释液1（DMSO）溶解、摇匀，配制成浓度为10.000 µmol/mL的Dxd标曲/质控储备液。

精密称取适量待测物B-1标准品于合适容器中，加入适量稀释液1（DMSO）溶解、摇匀，配制成浓度为10.000 µmol/mL的待测物B-1标曲/质控储备液。

精密称取适量内标A内标标准品于合适容器中，加入适量稀释液1（DMSO）溶解、摇匀，配制成浓度为1.000 mg/mL的内标储备液。

精密称取适量内标B (hydrochloride)内标标准品于合适容器中，加入适量储备液稀释液（DMSO）溶解、摇匀，配制成浓度为1.000 mg/mL的内标储备液。

工作液的配制

移取适量Dxd和待测物B-1的标曲储备液，加入稀释液2（ACN：H2O = 1:1，v/v）序列稀释至待测物A和待测物B（待测物B-1）的浓度分别为：30.000、60.000、150.000、600.000、3000.000、15000.000、24000.000、30000.000 pmol/mL的标曲工作液。

移取适量Dxd和待测物B-1的质控储备液，加入稀释液2（ACN：H2O = 1:1，v/v）序列稀释至Dxd的浓度为：90.000、900.000、9000.000、22500.000 pmol/mL的质控工作液。

移取适量内标A和内标B (hydrochloride)内标储备液，加入稀释液2（ACN：H2O = 1:1，v/v）稀释至内标的浓度分别为：25.000/100.000 ng/mL的内标工作液。

标曲/质控样品的配制

移取适量Dxd和待测物B-1的标曲工作液，加入空白基质分别稀释成待测物A和待测物B（待测物B-1）的浓度为：1.500、3.000、7.500、30.000、150.000、750.000、1200.000、1500.000 pmol/mL的标曲样品。

移取适量Dxd和待测物B-1的质控工作液，加入空白基质分别稀释成待测物A和待测物B（待测物B-1）的浓度为：4.500、45.000、450.000、1125.000 pmol/mL的质控样品。

方法学验证内容和接受标准

系统适用性

系统适用性考察是在分析批开始前进行以确保系统正常运转。处理至少1份系统适用性样品并连续进样至少3次，以最后有效的3次进样来评估系统适用性。

接受标准：在同一分析批中待测物及内标保留时间的变异系数（%CV）应不超出5.0%，待测物与内标色谱峰面积比值的变异系数（%CV）应不超出15.0%。

残留

残留通过在定量上限（ULOQ）样品后面进样双空白样品（不含待测物和内标）来进行评估。

接受标准：双空白样品中待测物保留时间处响应不超过同一分析批标曲符合接受标准的定量下限（LLOQ）待测物响应（均值）的20.0%，内标保留时间处响应不超过同一分析批符合接受标准的标准曲线定量下限内标响应（均值）的5.0%。

若残留无法满足接受标准，在方法学验证及样品分析过程中根据SOP-EXO-ABA-004对残留不满足接受标准批次的每一个样品受到上一个样品残留的影响进行评价，具体方法如下：

计算残留因子（若分析批中有多个，则选用最小的一个）：

残留因子 = 1 / （双空白样品中的峰面积/前一个ULOQ中的峰面积）

计算对下一个样品的影响率：

影响率 = 上一个样品的峰面积 / 残留因子 / 下一个样品的峰面积 * 100%

若影响率不高于5%，可认为实际发生的残留不会影响下一个样品的定量；若残留影响率大于5%，则说明下一个样品的定量结果受到了残留的影响，样品应进行相应处理：

质控样品：不论样品的准确度是否在85%~115%范围内，均按不符合接受标准计。

方法学验证考察项样品：该样品的结果不予采纳。若因残留评价失败导致考察项的统计数不够，则需重新做考察项。

若分析方法的残留不符合接受标准，可在分析方法中注明高浓度质控及ULOQ样品后需添加双空白样品。如有必要，可进一步优化分析方法解决残留问题。

标准曲线及范围

LLOQ作为样品中可被定量测定的最低浓度，已经在方法开发中进行了考察，满足预设的精密度和准确度。LLOQ浓度在3 ~ 5个半衰期时样品中的药物浓度，或Cmax的1/10～1/20时的药物浓度附近。待测物A和待测物B（待测物B-1）的标准曲线浓度范围为1.500 pmol/mL到1500.000 pmol/mL；标准曲线样品的浓度分别为：1.500、3.000、7.500、30.000、150.000、750.000、1200.000、1500.000 pmol/mL。以待测物浓度为横坐标，待测物与内标化合物色谱峰的峰面积比值为纵坐标，用加权最小二乘法进行回归运算（权重为1/X2），求得回归方程，即为标准曲线。

接受标准：

标准曲线各浓度点的回算值与标示值之间的偏差须在±15.0%以内（定量下限处的偏差在±20.0%范围内）。

标准曲线的R2不应低于0.9800。

75%的非零浓度点且至少有六个浓度点（包括LLOQ和ULOQ）用于标准曲线的计算，且满足以上标准。

每个浓度点含有2个标准曲线样品的情况下，必须满足每个浓度点至少50%的样品满足接受标准。

应至少在一次评估中采用新鲜配制的校正标样制备校准曲线。

批内准确度及精密度

取五个不同浓度的质控样品（待测物A和待测物B（待测物B-1）浓度分别为1.500 pmol/mL（定量下限）、4.500 pmol/mL（低）、45.000 pmol/mL（几何均值）、450.000 pmol/mL（中）和1125.000 pmol/mL（高）），考察批内准确度和精密度。每个分析批每个浓度平行处理6份，需新鲜配制准确度精密度样品。批内准确度和精密度通过每个分析批的质控样品进行评价。

报告的方法验证数据以及准确度和精密度的测定应包括所有获得的结果，包括不符合接受标准的单个质控样品，已被记录的明显错误情况除外。同时还要提交每个分析批的批内准确度和精密度数据。

接受标准：

每一浓度水平下批内准确度偏差应在±15.0%范围内（定量下限处的偏差在±20.0%范围内）。

每一浓度水平下批内精密度应不超过15.0%（LLOQ样品不超过20.0%）。

质控样品

每个分析批至少包含低（4.500 pmol/mL）、几何均值（45.000 pmol/mL）、中（450.000 pmol/mL）、高（1125.000 pmol/mL）4个浓度的质控样品（每个浓度水平至少2份）。

接受标准：每个分析批至少2/3的质控样品且每个浓度水平至少50%的样品准确度偏差应在±15.0%范围内。

稳定性

质控样品稳定性

考察质控样品在如下条件下的稳定性，随新鲜配制标准曲线样品一同测定：

短期稳定性：分别将低、高浓度的质控样品在冰水浴白光解冻后放置于冰水浴白光条件下（至少6小时，考察时间可根据实际情况调整）。各浓度水平平行测定6份。

接受标准：每个浓度的平均值的准确度应不超出85.0%~115.0%，且每个浓度的变异系数（%CV）应不大于15.0%。

验证考察项不符合接受标准的处理方法

批内准确度与精密度分析批不符合接受标准的处理方法

首先应分析失败可能的原因，如能明确原因（比如：配制错误、方法错误等），调整后重新考察；如未知原因导致批内准确度与精密度分析批失败，可继续考察但需要连续通过两个批内准确度和精密度分析批，如果下一个验证批次也失败，则应在重新启动验证之前修改该方法。

当出现失败时，可计算是否存在离群值。如果出现离群值（离群值的定义为超过除离群值之外的均值±3SD的范围外的数值），包括离群值在内的数据都需要报告。包含和剔除离群值的准确度和精密度的统计计算结果均需要报告。

稳定性考察不符合接受标准的处理方法

如果稳定性考察中其中一个或者两个浓度的测得浓度均值不符合接受标准，稳定性考察需要重复测定或者直接缩短考察时间。如果重复的稳定性考察仍然不能满足接受标准，报告质控样品在该条件下不稳定。可以考察更短的储存时间。如果第二次稳定性考察满足接受标准，需要进行第三次稳定性考察。如果第三次稳定性考察确认了第二次稳定性考察的结果，第三次稳定性考察的结果作为最终的稳定性数据进行报告。如果第三次稳定性考察不满足接受标准，报告稳定性考察失败。考察更短的储存时间。

如果稳定性考察中其中一个或者两个浓度的变异系数大于15.0%，稳定性考察可重复测定一次。
