系统适用性

系统适用性按验证方案考察，测定结果见表2，所有结果均符合接受标准，表明进样时检测仪器均处于正常运转状态。

残留

残留按验证方案测定，结果见表3。所有分析批结果均符合接受标准。

标准曲线及范围

标准曲线及范围按验证方案测定，方法验证中的各标准曲线回归方程及各标准样品浓度点测定结果见表4和表5，典型标准曲线见图1。结果表明，本方法测定BALB/c裸鼠血浆中待测物A和待测物B（待测物B-1）的线性范围可达1.500 ~1500.000 pmol/mL。

批内准确度及精密度

批内准确度及精密度按验证方案考察，根据质控样品测定结果计算准确度及精密度，结果见表6。

待测物A除LLOQ QC以外的所有浓度质控的批内准确度偏差范围为-0.7% - 0.9%，批内精密度最大值为9.4%。LLOQ QC的批内准确度偏差为-9.4%，批内精密度为8.5%。

待测物B（待测物B-1）除LLOQ QC以外的所有浓度质控的批内准确度偏差范围为2.1% - 6.6%，批内精密度最大值为3.2%。LLOQ QC的批内准确度偏差为-0.1%，批内精密度为5.7%。

结果表明，质控样品的批内精密度和准确度均符合接受标准。

质控样品

至少包含有低（4.500 pmol/mL）、几何均值（45.000 pmol/mL）、中（450.000 pmol/mL）、高（1125.000 pmol/mL）4个浓度的质控样品按验证方案测定，质控样品结果见表6。结果表明，所有接受批的质控样品均符合分析批接受标准。

质控样品稳定性

短期稳定性

短期稳定性按验证方案测定，测定结果见表7。结果表明，BALB/c裸鼠血浆样品中待测物A和待测物B（待测物B-1）可在冰水浴白光条件下稳定至少16小时。

结论

本实验对分析方法“LC-MS/MS方法测定BALB/c裸鼠EDTA-K2血浆样品中待测物A（RESS25255-B）和待测物B（待测物B-1）（RESS25008）的浓度”进行了部分验证。本实验成功验证了分析方法的系统适用性，残留，标准曲线及范围，批内准确度及精密度，样品短期稳定性。

综上，本方法可以用于测定BALB/c裸鼠血浆中的待测物A和待测物B（待测物B-1）的浓度。
